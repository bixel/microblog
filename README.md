# SOAK Skynet 2.0

Install the python deps
``` shell
pip install -r requirements.txt
```

Initialize the db and run the server (will listen on
[localhost:5000](http://localhost:5000))
``` shell
./setup_db.py
./blog.py
```

Install the javascript
``` shell
npm install
```

and finally build the app (and watch)
``` shell
npm start
```
